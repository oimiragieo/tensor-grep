1. **Add `pub path_was_implicit: bool` to `RipgrepSearchArgs`** (`rg_passthrough.rs:16-109`) ‚Äî a compiler-enforced ratchet: every exhaustive struct-literal site fails to compile until set. `#[derive(Default)]` ‚Üí `false` fail-safe for the one test helper (`:601`).
2. **Insert the check as the FIRST statement in `execute_ripgrep_search`** (before `resolve_ripgrep_binary()`): `if let Some(refusal) = check_implicit_walk_ceiling(args) { eprintln!("{refusal}"); return Ok(2); }`. **RETURN `Ok(2)`, NEVER `process::exit()`/`Err`/`bail!`** ‚Äî this is the load-bearing decision: the PyO3 FFI bridge (`lib.rs:332`) embeds rust_core, so `process::exit` would hard-kill the host Python; `Err`/`bail!` reintroduces the exit-1-vs-exit-2 no-match-ambiguity bug (audit #81 #7, `main.rs:5474-5479`). All 8 CLI call sites already handle `Ok(exit_code)` via `if exit_code != 0 { process::exit(exit_code.max(1)); }` ‚Üí **zero call-site edits for exit handling**.
3. **Module move:** the 3 probe items move from `main.rs` (binary crate root) ‚Üí `rg_passthrough.rs` (library), `pub(crate)`; `main.rs` re-imports for its existing tests. Flag the cross-module diff in the PR.
